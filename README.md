# 📱 TokoKu Mobile — Shopping Assistant Chatbot

Aplikasi mobile chatbot berbasis **React Native** yang berfungsi sebagai asisten belanja virtual untuk TokoKu. Pengguna dapat bertanya seputar produk yang dijual dalam Toko-ku.

---

## ✨ Features

-  **Real-time chat** dengan bot asisten belanja
-  **Multi-session** — buat, ganti, dan hapus sesi chat
-  **Edit pesan** — edit pesan yang sudah terkirim dan otomatis dikirim ulang
-  **History panel** — lihat semua sesi chat beserta judulnya


## 🛠️ Tech Stack
- React Native (0.81.5)
- React (19.1.0)



## 🔌 Backend Configuration

## 🚀 Getting Started

Prerequisites: 
- Node.js v18+
- Expo CLI
- Android emulator / iOS simulator, atau aplikasi Expo Go di perangkat fisik

Installation
```bash
npm install
```
Running
```bash
npm run android  # Android
npm run ios      # iOS
npm run web      # Web
```
